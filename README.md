# AI Security Policy Chatbot

[![Live App](https://img.shields.io/badge/Live-Demo-green)](https://your-link.streamlit.app)

## Features
- Upload PDF
- Ask questions
- AI answers from document
- Shows sources
- Chat UI

##  Tech Stack
- Streamlit
- LangChain
- HuggingFace
- ChromaDB

##  Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
```
